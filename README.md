Conversational chat-bot for Delta Chat, using GPT4All.

## Install

```sh
pip install deltachat-chatbot
```

## Usage

Configure the bot:

```sh
chatbot init bot@example.com PASSWORD
```

Start the bot:

```sh
chatbot serve
```

Run `chatbot --help` to see all available options.
